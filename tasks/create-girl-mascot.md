# 任务：将右下角 AI 对话入口改为二次元小女孩形象

## 背景
当前 WebWiki 桌面端右下角的 AI 对话入口是一个默认的蓝色圆形气泡按钮。需要将其替换为一个二次元风格的小女孩形象入口，增强产品亲和力。

## 需求
1. 替换元素：右下角固定的 AI 对话入口按钮。
2. 形象要求：
   - 二次元风格小女孩，类似 QQ 秀。
   - 穿裙子。
   - 裙子上的花纹使用项目 logo：`assets/images/logo.png`（DNA 双螺旋图案）作为纹理/图案填充。
   - 大小统一后不再调整，建议默认显示尺寸为 80x100px 左右。
3. 动画要求：
   - 必须有轻微循环动画，例如眨眼、裙摆轻微飘动、或呼吸感缩放。
   - 鼠标悬停时可以有额外反馈（如轻微放大、招手）。
4. 功能保留：
   - 点击小女孩入口后，仍然打开/关闭 AI 对话窗口。
   - 保持当前对话按钮的定位逻辑（固定在右下角，不影响页面布局）。

## 参考文件
- Logo：`assets/images/logo.png`
- 当前入口样式：`css/style.css` 中搜索 `.chat-toggle` / `#chatToggle`
- 当前入口 HTML/JS：`index.html` 和 `js/chat.js`

## 实现建议
- 推荐使用 SVG 绘制小女孩，便于缩放和动画控制。
- 裙子花纹使用 CSS `mask-image` 或 SVG `<pattern>` 引用 `assets/images/logo.png`。
- 动画使用 CSS `@keyframes`，避免使用 JavaScript 高频重绘。
- 如需使用外部图片，优先使用项目内已有资源，不要依赖网络资源。

## Reviewer 职责
在修复完成后，必须承担 Reviewer 角色：
1. 检查 CSS/JS 是否存在语法错误。
2. 确认小女孩入口在不同页面（首页、任务、项目、设置）都能正常显示。
3. 确认点击后仍能正常打开 AI 对话窗口。
4. 确认没有破坏其他样式或布局（特别是设置页面）。
5. 不要修改与本任务无关的代码。

## 验收标准
1. 右下角显示二次元小女孩形象，而不是蓝色气泡按钮。
2. 裙子上有 logo.png 的 DNA 双螺旋图案作为花纹。
3. 小女孩有循环动画（眨眼/裙摆飘动/呼吸等）。
4. 点击小女孩能打开 AI 对话窗口。
5. 设置页面和其他页面样式没有被破坏。
